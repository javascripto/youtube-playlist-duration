# Privacy Policy — YouTube Playlist Duration

Last updated: March 2, 2026

## Summary
The **YouTube Playlist Duration** extension calculates and displays total YouTube playlist duration directly in the playlist title.

## Data Collection
This extension does **not collect**, **does not store on external servers**, and **does not sell** personal data.

## Data Accessed Locally
To work properly, the extension may access:
- YouTube page content required to read visible video durations from the playlist.
- User display preference (prefix/suffix), stored in `chrome.storage.sync`.

## Permissions Usage
- `host_permissions` for YouTube pages: used only to run the content script on matching YouTube pages.
- `storage`: used only to save extension preferences.
- `activeTab`: used for extension operation in the active tab context.

## Data Sharing
The extension does not share data with third parties.

## Data Retention
The extension keeps only local preference data while installed (or synchronized by the browser account, when supported).

## Security
All processing is done locally in the user's browser.

## Children
This extension is not specifically directed to children.

## Changes to This Policy
This policy may be updated to reflect extension changes. The "Last updated" date will be revised when applicable.
