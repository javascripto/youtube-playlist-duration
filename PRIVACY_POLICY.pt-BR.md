# Política de Privacidade — YouTube Playlist Duration

Última atualização: 2 de março de 2026

## Resumo
A extensão **YouTube Playlist Duration** calcula e exibe a duração total de playlists do YouTube diretamente no título da playlist.

## Coleta de dados
Esta extensão **não coleta**, **não armazena em servidores externos** e **não vende** dados pessoais.

## Dados acessados localmente
Para funcionar corretamente, a extensão pode acessar:
- Conteúdo da página do YouTube necessário para ler as durações visíveis dos vídeos da playlist.
- Preferência do usuário para exibição (prefixo/sufixo), salva em `chrome.storage.sync`.

## Uso de permissões
- `host_permissions` para páginas do YouTube: usada apenas para executar o content script nas páginas compatíveis.
- `storage`: usada apenas para salvar preferências da extensão.


## Compartilhamento de dados
A extensão não compartilha dados com terceiros.

## Retenção de dados
A extensão mantém apenas dados locais de preferência enquanto estiver instalada (ou sincronizados pela conta do navegador, quando suportado).

## Segurança
Todo o processamento é feito localmente no navegador do usuário.

## Crianças
Esta extensão não é direcionada especificamente a crianças.

## Alterações nesta política
Esta política pode ser atualizada para refletir mudanças na extensão. A data de "Última atualização" será revisada quando necessário.
