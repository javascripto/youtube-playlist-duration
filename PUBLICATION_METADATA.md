# Publication Metadata

## Extension Name
YouTube Playlist Duration

## Short Description
Shows total YouTube playlist duration directly in the playlist title.

## Full Description
YouTube Playlist Duration automatically calculates the sum of visible videos in a YouTube playlist and displays the total in the playlist title.

Main features:
- Automatic updates when playlist content changes.
- Display mode option: prefix or suffix.
- Works in Chrome- and Firefox-based browsers.

Known limitation:
- The total includes videos currently loaded in the playlist DOM (practical limit around ~200 items on YouTube). Items not loaded by the UI are not included.
